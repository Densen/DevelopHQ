Interaction Design

User interfaces & application architecture is evolving

These days you are more likely to be creating a mobile or desktop app than a website. On the server side it's all about the creation of great APIs, on the client side it's all about creating great experiences through interaction design.

Prototype Creation

Prototypes can be used for a number of reasons

They can validate a business idea, scope out the application or test and gain user feedback. They are always useful, quick to create and a must for every web app.

We use a number of different tools to create prototypes but often our favourite is good old HTML which has the added bonus of forming the basis of your application build, saving time & money.
